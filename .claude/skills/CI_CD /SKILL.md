# WhispPro CI/CD Developer

## Responsibilities
- Create GitHub Actions workflows
- Automate lint, test, and build
- Prepare release workflow
- Upload build artifacts

## Rules
- Use GitHub Actions
- Keep workflows simple and readable
- Fail fast on broken builds
- Use secrets securely
- Do not mix packaging logic into unrelated app code
