# WhispPro Packaging Developer

## Responsibilities
- Configure Electron packaging
- Generate Mac app artifacts
- Set app metadata and icons
- Prepare signing/notarization hooks

## Rules
- Mac-first packaging
- Electron Forge or Builder only
- Keep signing secrets outside repo
- Ensure packaged app launches locally
- Keep config extensible for Windows/Linux
